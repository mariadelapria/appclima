# Programa de previsão do tempo com interface interativa usando Streamlit e API OpenWeatherMap

O AppClima é um aplicativo desenvolvido em Python com Streamlit, que permite consultar o clima de qualquer cidade do mundo. 
Ele utiliza a API do OpenWeatherMap para obter dados atualizados e exibe a previsão dos próximos 5 dias, incluindo informações sobre temperatura, clima, umidade e velocidade do vento.

O app também oferece análises avançadas de temperatura usando Pandas, como média semanal, picos de calor e baixas de temperatura.
Para visualização, utiliza Plotly, gerando gráficos interativos e claros que facilitam a interpretação dos dados.

A interface foi personalizada com CSS e ícones próprios, proporcionando uma experiência visual moderna, intuitiva e agradável. 
Ao digitar o nome da cidade, os dados são atualizados automaticamente, permitindo explorar o clima de diferentes regiões em tempo real.

As principais tecnologias e recursos utilizados neste projeto incluem:

Python 3

Streamlit para interface web interativa

OpenWeatherMap API para dados climáticos

Pandas para manipulação e análise de dados

Plotly para gráficos interativos

CSS e ícones personalizados para estilização

Funcionalidade de previsão para os próximos 5 dias

Análise de temperatura: média, pico e baixa da semana
